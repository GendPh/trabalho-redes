Objetivo deste projeto

Este projeto tem como objetivo a criação de um sistema de redes numa empresa fictícia chamada PWM RComputadores.

A empresa PWM RComputadores contém uma livraria aberta ao público. Contém também um espaço reservado compartilhado ou reservado, onde os clientes registados podem acessar a internet, fazer pesquisas, trabalhos, etc.

Ferramenta

Para a realização deste projeto foi utilizado o software Cisco Packet Tracer para simular a rede a instalar.

Para tal a planta do espaço foi providenciada pelo cliente, bem como os equipamentos predefinidos a ser utilizados

Tipos Equipamentos

Equipamentos ativos

Importantes para a construção da infraestrutura, permitindo que todos os dados sejam encaminhados de forma eficiente e segura entre os dispositivos conectados, garantindo assim uma comunicação fluída dentro do ambiente da infraestrutura.

Os equipamentos utilizados nesta categoria foram:

  • Router
  • Servidores
  • Switch
  • Access Point

Equipamentos passivos

Essenciais na construção da infraestrutura, onde desempenham a conectividade e transmissão de dados entre os dispositivos

Os equipamentos utilizados nesta categoria foram:

  • Cabos rede
